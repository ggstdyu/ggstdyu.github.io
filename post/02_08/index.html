不安全的脚本

:-)

Chrome 弹出提示：已拦截不安全内容，此网页正试图从未经验证的来源加载脚本。

解决办法：查看拦截内容详情，键入 F12，切至 Console 选项卡，其中 Mixed Content 所示即为不安全的脚本。一般情况是 HTTPS 网页中插入了 HTTP 链接，全部改为 HTTPS 后拦截消失。

深似海的 GitHub

:-)

Git 是目前世界上最先进的分布式版本控制系统。GitHub 是支持 Git 作为唯一的版本库格式且面向开源及私有软件项目的托管平台，其上托管了无数优秀的开源项目。作为计算机的小学生，好好利用 GitHub 并参与合适的开源项目是提升能力的重要途径。

明天的日程围绕 GitHub 上有趣的项目展开。

下乘的麻将

:-)

知乎上关于麻将的话题精华中有这些问题：

  • 年轻人打麻将是浪费人生吗？
  • 同为博弈为什么围棋高雅，而麻将电子竞技被视为低俗？
  • 如何让妈妈远离麻将桌？不是让她不玩，只是想和她经历更多可以快乐的事情？

不管啦，不因为围棋被视高雅才学习围棋，不因为麻将被视低俗而厌恶麻将，都只是生活的乐趣罢。享受生活的方式人人不同，爱旅游的行万里路，爱看字的读万卷书，找到人生的乐趣便好。有风打风，没风幺九冲；金三银七，打高不打低。
