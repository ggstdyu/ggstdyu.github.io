深度学习 Markdown

:-)

Markdown 好麻烦啊啊啊！
就换个行，又是空格又是回车又是制表符，头都大了。

另有一个问题就是，中文与西文混排时是否要加入空格。参考众多说法后，同意混排时不一定要有空格字符，但一定要有间距。考虑到增加 Markdown 文本的易读性，最终决定插入空格。参考：
https://www.zhihu.com/question/19587406

开启 Double Commander

:-)

觉得 Finder 难用得很，搜索一番选定 Double Commander。其实在另一台 Windows10 平台上安装的是 XYplorer，但要支持 Mac 平台就筛去很多优秀的软件。参考：
https://alternativeto.net/software/finder/?platform=mac

启动之后感觉有些丑。。。在界面优雅方面跟 Finder 一比简直像上世纪的软件了。。。先试用几天吧，起码它支持分栏和多标签，还有正常的剪切功能和排序功能。。。
